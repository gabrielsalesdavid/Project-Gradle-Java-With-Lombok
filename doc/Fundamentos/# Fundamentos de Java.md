# Fundamentos de Java

## Introdução
Java é uma linguagem de programação orientada a objetos, robusta e multiplataforma, que segue o princípio "Write Once, Run Anywhere" (WORA).

## Conceitos Básicos

### 1. Tipos de Dados
- **Primitivos**: int, long, float, double, boolean, char, byte, short
- **Referência**: String, Arrays, Objetos

### 2. Orientação a Objetos
- Classes e Objetos
- Encapsulamento
- Herança
- Polimorfismo
- Abstração

### 3. Estruturas de Controle
- Condicionais: if, else, switch
- Loops: for, while, do-while
- Exceções: try, catch, finally

### 4. Coleções
- List, Set, Map
- ArrayList, HashMap, HashSet

## Ferramentas Utilizadas neste Projeto
- **Lombok**: Reduz código boilerplate com annotations
- **MapStruct**: Mapping entre objetos Java
- **Gradle**: Build automation
